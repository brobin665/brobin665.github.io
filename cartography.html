Cartographic Projects

Longform Infographic

Longform Infographic, Geog 572, UW-Madison

The purpose of this lab was to design a longform infographic that tells a compelling story, as well make use of a bivariate map. I choose to tell the story of the gas station brand,Kwik Trip, and compare the number of Kwik Trip gas stations in Wisconsin to other gas station brands. In order to complete this lab, I used Adobe Illustrator to design the layout, Microsoft Excel to modify datasets and create a graph, and ArcMap to create the bivariate map. There are a few ways to improve this infographic. First off, I can use icons and images to replace some of the text in the lower half of the inforgraphic. Secondly, I could simplify the layout, by removing either the white boxes behind the text, or the red bars defining sections, which serve the same purpose, and just add visual complexity to the infographic, without providing any new information.

Digital Terrain Representation

Digital Terrain Representation, Geog 572, UW-Madison

The purpose of this lab was to create a bike route for a hypothetical international cycling union, and to map that route with representation of the terrain along the route. I recently took a road trip out to the western United States, and used several stops from my own trip as inspiration for this route. Software I used included Google Maps, to create the route, ArcMap to create the file to hold the route from Google Maps, as well as to modify and collect the landcover, DEM, and satelite data, Adobe Photoshop to create the shaded relief layer, and Adobe Illustrator to create the layout for the map. To further enhance this project, I could move the legend, and the list of stops along the route over the corners of the map which don't include stops along the routes.

Aesthetic Style

Aesthetic Style Applied to a Tileset, Geog 572, UW-Madison

The purpose of this lab was to apply a specific style to a set of web tiles. The theme of this map is "Alien Invasion!", inspired by artwork from "Mars Attack!", and the video games "Missile Defense" and "Destroy All Humans!" . I used Adobe Photoshop to create some of the textures used, as well as Adobe Illustrator to design the alien UFO icons. Primarily, I used Mapbox Studio and CartoCSS to design and style this tileset.
